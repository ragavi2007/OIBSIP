# OIBSIP-L2ToDoWebApp

## To-Do Web App

### Project Overview
This project is a fully interactive **To-Do List Web Application** built using HTML, CSS, and Vanilla JavaScript. It helps users manage daily tasks efficiently by organizing them into **Pending** and **Completed** sections.

The app allows users to add, edit, complete, and delete tasks with a clean and responsive user interface.

---

## Features

- Add new tasks using input field and button
- Tasks appear instantly in the Pending list
- Mark tasks as complete and move them to Completed list
- Edit tasks inline without page refresh
- Delete tasks permanently from both lists
- Dynamic task counters for Pending and Completed tasks
- Timestamp for each task (created and completed time)
- Empty state messages for better user experience
- Responsive design for all devices
- Data persistence using localStorage (tasks remain after refresh)

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
