# OIBSIP-L1-TributePage

## Tribute Page - Dr. A.P.J. Abdul Kalam

### Project Overview
This project is a visually engaging Tribute Page dedicated to **Dr. A.P.J. Abdul Kalam**, the renowned Indian scientist and former President of India.

The webpage highlights his inspiring life journey, scientific achievements, contributions to India's space and missile programs, and his legacy as a great teacher and leader.

## Features

- Hero section with Dr. A.P.J. Abdul Kalam's name and tagline
- Tribute image using a royalty-free image source
- Biography section with original written content
- Timeline showcasing major achievements and milestones
- Inspirational quote section
- Responsive design for desktop, tablet, and mobile devices
- Multiple background sections with modern styling
- Different font styles for headings and body content
- Clean and user-friendly interface

## Technologies Used

- HTML5
- CSS3
